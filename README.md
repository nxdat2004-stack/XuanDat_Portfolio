## 🚀 How to host on GitHub Pages

### Step 1 — Create a GitHub repo

1. Go to [github.com](https://github.com) → **New repository**
2. Name it: `portfolio` (or anything you want)
3. Set to **Public**
4. Click **Create repository**

### Step 2 — Upload files

1. Inside the new repo, click **Add file → Upload files**
2. Drag and drop **all files in this folder** (index.html, support.js, image-slot.js, and all images)
3. Click **Commit changes**

### Step 3 — Enable GitHub Pages

1. Go to repo **Settings** → **Pages** (left sidebar)
2. Under **Branch**, select `main` → `/ (root)`
3. Click **Save**
4. Wait ~1 minute

### Step 4 — Your site is live!

Your URL will be:
```
https://YOUR-USERNAME.github.io/REPO-NAME/
```

Example: `https://nxdat.github.io/portfolio/`

## 📁 File structure

```
index.html        ← Main portfolio page
support.js        ← Runtime (required)
image-slot.js     ← Image component (required)
*.jpg / *.png     ← All portfolio images (76 files)
```

**Important:** Keep all files in the same folder. Do not move images into subfolders — the portfolio references them by name from the root.

---

## ✏️ Updating content

To update the portfolio, edit `index.html` directly in GitHub (click the file → pencil icon) or re-upload a new version.
